# no-lonely-if

Discourage if statements as the only statement in else blocks

# Examples

❌ Incorrect

if (foo) {
    // ...
} else {
    if (bar) {
        // ...
    }
}

if (condition) {
    // ...
} else {
    if (anotherCondition) {
        // ...
    }
}

✅ Correct

if (condition) {
    // ...
} else if (anotherCondition) {
    // ...
} else {
    // ...
}
