# no-extra-semicolons

Disallow extra semicolons.

# Examples

❌ Incorrect

a {
  color: #000000;;
}

a {
  ;color: #ffffff;
}

a {
  color: #ffffff;
  ;
}

✅ Correct

a {
    color: #ffffff;
}
