# vue/html-closing-bracket-spacing warning

This rule aims to enforce consistent spacing style before closing brackets > of tags.

# Examples

❌  Incorrect

<template>
    <div>
        <div ></div >
        <div foo ></div >
        <div foo="bar" ></div >
        <div/>
        <div foo/>
        <div foo="bar"/>
    </div>
</template>

✅  Correct

<template>
    <div>
        <div></div>
        <div foo></div>
        <div foo="bar"></div>
        <div />
        <div foo />
        <div foo="bar" />
    </div>
</template>
