# property-case warn

Specify lowercase for properties.

# Examples

❌ Incorrect

a {
    Width: 1px
}

a {
    WIDTH: 1px
}

a {
    border-Radius: 5px;
}

a {
    -WEBKIT-animation-duration: 3s;
}

@media screen and (orientation: landscape) {
    WiDtH: 500px;
}

✅ Correct

a {
    width: 1px
}

a {
    border-radius: 5px;
}

a {
    -webkit-animation-duration: 3s;
}

@media screen and (orientation: landscape) {
    div {
        width: 500px;
    }
}
