# number-leading-zero warn

Requires a leading zero on fractional values that are less than one.

# Examples

Incorrect

a {
    line-height: .5px;
}

a {
    transform: translate(2px, .4px);
}

Correct

a {
    line-height: 0.5px;
}

a {
    transform: translate(2px, 0.4px);
}
