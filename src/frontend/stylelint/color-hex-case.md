# color-hex-case

> Sets the case of hex values to lowercase.


## Examples

<code-highlight>
 
<div slot="correct">

```css
a {
    color: #ffffff;
}

a {
    color: #ff0000;
}
```

</div>

 
<div slot="incorrect">

```css
a {
    color: #FFF;
}

a {
    color: #FF0000;
}
```

</div>

 
</code-highlight>
