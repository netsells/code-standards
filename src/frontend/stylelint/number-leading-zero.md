# number-leading-zero

> Requires a leading zero on fractional values that are less than one.


## Examples

<code-highlight>
 
<div slot="correct">

```css
a {
    line-height: 0.5px;
}

a {
    transform: translate(2px, 0.4px);
}
```

</div>

 
<div slot="incorrect">

```css
a {
    line-height: .5px;
}

a {
    transform: translate(2px, .4px);
}
```

</div>

 
</code-highlight>
