# color-named

> Colors must never be named.

 

## Examples

<code-highlight>
 
<div slot="correct">

```css
a { 
    color: #000000; 
}

a {
    color: $blue; 
}
```

</div>

 
<div slot="incorrect">

```css
a { 
    color: black; 
}

a {
    color: white; 
}
```

</div>

 
</code-highlight>
