# length-zero-no-unit

> Zero lengths should not have units.

 

## Examples

<code-highlight>
 
<div slot="correct">

```css
.my-element {
    top: 0;
    left: 0;
}
```

</div>

 
<div slot="incorrect">

```css
.my-element {
    top: 0px;
    left: 0px;
}
```

</div>

 
</code-highlight>
