# no-multiple-empty-lines

> Restricts the number of allowed empty lines

 

## Examples

<code-highlight>
 
<div slot="correct">

```js
const foo = bar;

const bar = foo;
```

</div>

 
<div slot="incorrect">

```js
const foo = bar;



const bar = foo;
```

</div>

 
</code-highlight>
