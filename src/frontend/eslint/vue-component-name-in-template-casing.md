# vue/component-name-in-template-casing

> Enforce specific casing for the component naming style in template
 

## Examples

<code-highlight>
 
<div slot="correct">

```vue
<template>
    <my-component />
</template>
```

</div>

 
<div slot="incorrect">

```vue
<template>
    <myComponent />
    <MyComponent />
    <my_component />
</template>
```

</div>

 
</code-highlight>
