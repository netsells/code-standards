# vue/script-indent

> Enforce consistent indentation in ```<script>```
 

## Examples

<code-highlight>
 
<div slot="correct">

```vue
<script>
    let a = {
        foo: 1,
        bar: 2
    }
</script>
```

</div>

 
<div slot="incorrect">

```vue
<script>
let a = {
    foo: 1,
    bar: 2
}
</script>
```

</div>

 
</code-highlight>
