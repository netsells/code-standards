# vue/mustache-interpolation-spacing

> Expect one space between expression and curly brackets.

 

## Examples

<code-highlight>
 
<div slot="correct">

```vue
<div>{{ foo }}</div>
```

</div>

 
<div slot="incorrect">

```vue
<div>{{foo}}</div>
<div>{{    foo    }}</div>
```

</div>

 
</code-highlight>
