# vue/key-spacing

> Enforce consistent spacing between keys and values in object literal properties
 

## Examples

<code-highlight>
 
<div slot="correct">

```vue
<template>
    <span>Message: {{ msg }}</span>
</template>
```

</div>

 
<div slot="incorrect">

```vue
<template>
    <span>Message : {{ msg }}</span>
    <span>Message :{{ msg }}</span>
</template>
```

</div>

 
</code-highlight>
