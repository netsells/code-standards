# quotes

> Enforce the use of single quotes when using JavaScript

 

## Examples

<code-highlight>
 
<div slot="correct">

```js
const array = [];
const array = [ 'red', 'green' ];
const array = new Array(23);
```

</div>

 
<div slot="incorrect">

```js
const array = [ , , ];
const array = [ 'red', , 'green'];
```

</div>

 
</code-highlight>
